**Official Layout Structure:**

1. **Header Block**
   - Agency name: TEXAS COMPTROLLER OF PUBLIC ACCOUNTS
   - Form number in bold box
   - Form title

2. **Section 1: Basic Information**
   - Line 1: Taxing Unit Name
   - Line 2: County
   - Line 3: Tax Year
   - Line 4: Disaster Area Checkbox

3. **Section 2: Last Year's Levy Information**
   - Line 5: Last Year's Total Levy ($)
   - Line 6: Last Year's M&O Rate ($)
   - Line 7: Last Year's Debt Rate ($)
   - Line 8: Lost Property Levy ($)

4. **Section 3: Current Year's Property Values**
   - Line 9: Current Total Taxable Value ($)
   - Line 10: New Property Value ($)
   - Line 11: Tax Increments/TIF ($)

5. **Section 4: Proposed Tax Rate**
   - Line 12: Proposed M&O Rate ($)
   - Line 13: Proposed Debt Rate ($)
   - Line 14: Total Outstanding Debt ($)

6. **Section 5: Calculated Tax Rates** (Highlighted Box)
   - No-New-Revenue Tax Rate (Calculated)
   - Voter-Approval Tax Rate (Calculated)
   - De Minimis Rate (Calculated)
   - Proposed Rate (Calculated)

7. **Certification Section**
   - Statement of accuracy
   - Signature line
   - Date and title fields
